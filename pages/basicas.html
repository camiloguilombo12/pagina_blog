Etiquetas Básicas de la Estructura del Documento

<!DOCTYPE html>

Indica al navegador que el documento está escrito en HTML5. Siempre debe ir en la primera línea del archivo y es fundamental para que el navegador interprete correctamente el código.


<!DOCTYPE html>

<html>

Es la etiqueta raíz del documento. Todo el contenido de la página debe estar dentro de esta etiqueta. También puede incluir el atributo lang para indicar el idioma del sitio.


<html lang="es">
</html>

<head>

Contiene información interna del documento que no es visible para el usuario, como configuraciones, metadatos, enlaces a archivos CSS y scripts.


<head>
</head>

<title>

Define el título de la página que aparece en la pestaña del navegador. Es importante para la identificación del sitio y el SEO.


<title>Mi Página Web</title>

<body>

Contiene todo el contenido visible del sitio web, como textos, imágenes, enlaces, formularios y demás elementos que el usuario puede ver e interactuar.


<body>
</body>

Etiquetas Recomendadas (Estructura Semántica)

<header>

Representa el encabezado de una página o sección. Generalmente contiene el título principal, logotipo o información introductoria.


<header>
</header>

<nav>

Se utiliza para agrupar los enlaces de navegación del sitio, como menús principales o secundarios.


<nav>
</nav>

<main>

Define el contenido principal del documento. Solo debe existir una etiqueta <main> por página y no debe contener elementos repetitivos como menús.


<main>
</main>

<section>

Agrupa contenido relacionado bajo un mismo tema. Se utiliza para dividir el contenido en bloques organizados.


<section>
</section>

<article>

Representa contenido independiente que podría existir por sí solo, como una noticia, publicación de blog o producto.


<article>
</article>

<aside>

Contiene información complementaria o secundaria relacionada con el contenido principal, como barras laterales o recomendaciones.


<aside>
</aside>

<footer>

Representa el pie de página del documento o de una sección. Generalmente incluye información como autor, derechos de autor o enlaces adicionales.


<footer>
</footer>
