4. Listas

<ul>

Define una lista desordenada. Los elementos se muestran con viñetas (puntos). Se utiliza cuando el orden de los elementos no es importante.

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>

<ol>

Define una lista ordenada. Los elementos se muestran numerados o con letras. Se utiliza cuando el orden sí es importante.

<ol>
    <li>Primer paso</li>
    <li>Segundo paso</li>
</ol>

<li>

Representa un elemento dentro de una lista. Se utiliza dentro de las etiquetas <ul> o <ol>.

<li>Elemento de lista</li>

<dl>

Define una lista de descripción. Se utiliza para mostrar términos y sus respectivas definiciones.

<dl>
    ...
</dl>

<dt>

Representa el término o concepto dentro de una lista de descripción. Va dentro de la etiqueta <dl>.

<dt>HTML</dt>

<dd>

Define la descripción o explicación del término indicado en <dt>. También se utiliza dentro de la etiqueta <dl>.

<dd>Lenguaje de marcado para crear páginas web.</dd>
