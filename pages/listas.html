Listas

<ul> (Lista desordenada)

Define una lista con viñetas. Se utiliza cuando el orden de los elementos no es importante.

  • Manzana
  • Banano
  • Naranja

<ul>
    <li>Manzana</li>
    <li>Banano</li>
    <li>Naranja</li>
</ul>

<ol> (Lista ordenada)

Define una lista numerada. Se usa cuando el orden sí es importante, como pasos o instrucciones.

  1. Paso uno
  2. Paso dos
  3. Paso tres

<ol>
    <li>Paso uno</li>
    <li>Paso dos</li>
    <li>Paso tres</li>
</ol>

<li> (Elemento de lista)

Representa un elemento dentro de una lista. Solo puede utilizarse dentro de <ul> o <ol>.

  • Ejemplo de elemento

<li>Elemento de lista</li>

<dl> (Lista de descripción)

Define una lista que contiene términos y sus definiciones.

HTML
Lenguaje de marcado para crear páginas web.

<dl>
    <dt>HTML</dt>
    <dd>Lenguaje de marcado para crear páginas web.</dd>
</dl>

<dt> (Término)

Representa el término o concepto dentro de una lista de descripción.

CSS
Lenguaje para dar estilo a páginas web.

<dt>CSS</dt>

<dd> (Descripción)

Define la descripción o explicación del término indicado en <dt>.

JavaScript
Lenguaje de programación para páginas web dinámicas.

<dd>Descripción del término</dd>
