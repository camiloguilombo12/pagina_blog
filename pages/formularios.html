Etiquetas de formularios

<form>

La etiqueta <form> define un formulario y agrupa todos los campos que el usuario puede rellenar y enviar.

Debajo tienes un formulario completo de ejemplo que usaremos para las demás etiquetas.

<label>

<label> asocia un texto descriptivo a un campo del formulario. El atributo for debe coincidir con el id del campo.

Ejemplo (usado en el formulario de arriba):


<label for="nombre">Nombre:</label>
<input id="nombre" type="text">

<input>

<input> crea diferentes tipos de campos (texto, email, contraseña, etc.) según el valor del atributo type.

Ejemplos reales del formulario:


<input type="text" id="nombre" name="nombre">
<input type="email" id="correo" name="correo">

<textarea>

<textarea> se usa para textos largos de varias líneas, como mensajes o comentarios.

Ejemplo del formulario:


<textarea id="mensaje" name="mensaje" rows="4">...</textarea>

<select> y <option>

<select> crea una lista desplegable y <option> define cada una de las opciones.

Ejemplo del formulario:


<select id="tema" name="tema">
    <option value="">-- Selecciona una opción --</option>
    <option value="soporte">Soporte técnico</option>
    <option value="ventas">Ventas</option>
    <option value="otros">Otros</option>
</select>

<button>

<button> crea un botón. Con type="submit" envía el formulario.

Ejemplo del formulario:


<button type="submit">Enviar formulario</button>

<fieldset> y <legend>

<fieldset> agrupa campos relacionados dentro del formulario y <legend> pone un título a ese grupo.

Ejemplo del formulario:


<fieldset>
    <legend>Datos de contacto</legend>
    ... campos del formulario ...
</fieldset>
