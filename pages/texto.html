Etiquetas de texto

<h1> a <h6>

Estas etiquetas se usan para títulos y subtítulos. Existen <h1>, <h2>, <h3>, <h4>, <h5> y <h6>, donde <h1> es el título más importante y <h6> el menos importante.

Ejemplo: <h1>Título principal</h1>

<p>

La etiqueta <p> se utiliza para definir un párrafo de texto.

Ejemplo: <p>Este es un párrafo de ejemplo.</p>

<span>

<span> se usa para agrupar pequeñas partes de texto dentro de una línea, normalmente para aplicar estilos o clases específicas.

Ejemplo: <p>Texto con <span>una parte resaltada</span> dentro.</p>

<strong>

<strong> indica que el texto es importante y normalmente se muestra en negritas.

Ejemplo: <p>Este texto es <strong>muy importante</strong>.</p>

<em>

<em> se usa para enfatizar texto, generalmente se muestra en cursiva.

Ejemplo: <p>Este texto está <em>enfatizado</em>.</p>

<br>

<br> inserta un salto de línea dentro del texto sin iniciar un nuevo párrafo.

Ejemplo: <p>Primera línea<br>Segunda línea</p>

<hr>

<hr> dibuja una línea horizontal que suele usarse para separar secciones de contenido.

Ejemplo: <p>Texto antes de la línea</p><hr><p>Texto después de la línea</p>
