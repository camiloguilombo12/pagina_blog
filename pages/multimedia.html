Etiquetas multimedia

<img>

La etiqueta <img> se usa para mostrar imágenes en la página. Usa el atributo src para indicar la ruta de la imagen y alt para una descripción alternativa.

Dónde poner el enlace de la imagen: en el atributo src.

Ejemplo real:

Ejemplo en código:


<img src="media/image.jpg" alt="Descripción de la imagen" width="300">

<audio> y <source>

<audio> se usa para reproducir archivos de audio. Con el atributo controls se muestran los controles de reproducción. Dentro se usa <source> para indicar el archivo de audio.

Dónde poner el enlace del audio: en el atributo src de <source>.

Ejemplo real:

Ejemplo en código:


<audio controls>
    <source src="media/audio.mp3" type="audio/mpeg">
</audio>

<video> y <source>

<video> se usa para mostrar videos. Con el atributo controls se muestran los controles (play, pausa, volumen, etc.). Dentro se usa <source> para indicar el archivo de video.

Dónde poner el enlace del video: en el atributo src de <source>.

Ejemplo real:

Ejemplo en código:


<video width="400" controls>
    <source src="media/video.mp4" type="video/mp4">
</video>

<figure> y <figcaption>

<figure> agrupa contenido multimedia (como imágenes o videos) y <figcaption> añade un pie de foto o descripción.

Dónde poner el enlace de la imagen/video: igual que en <img> o <video>, en el atributo src.

Ejemplo real:

Ejemplo en código:


<figure>
    <img src="media/image2" alt="Ejemplo de figure" width="300">
    <figcaption>Esta es una imagen con pie de foto.</figcaption>
</figure>
